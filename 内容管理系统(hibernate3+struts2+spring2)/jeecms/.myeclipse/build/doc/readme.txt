欢迎使用JEECMS v2.3.1 beta版，请按照以下步骤进行软件安装：

一、安装运行环境（JDK5+TOMCAT5.5+MYSQL5及以上，具体安装过程请参考本下载包中的相关说明文档）;

二、将程序解压后的JeeCms文件夹拷贝到tomcat安装目录下的webapps文件夹下（例如：D:\Tomcat6\webapps\），启动tomcat，在地址栏上输入http://localhost:8080/jeecms,请根据安装向导填写安装信息，安装完成后重启tomcat;

三、在浏览器的地址栏中输入http://localhost:8080/jeecms，若系统正常显示默认的模板页面则表示本系统顺利安完成.


附：
	系统管理后台登录：http://localhost:8080/jeecms/login/Jeecms.do
